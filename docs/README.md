# Documentation Index

Use this index to move quickly between phases and operational guides.

## New Contributor Quickstart

1. Run `cargo build --workspace`
2. Run `cargo test --workspace`
3. Run `cargo run -p hello_cli -- --name "world"`
4. Read [00 - Getting Started](00-getting-started.md)
5. Pick one phase and complete it end-to-end
